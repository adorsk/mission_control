class mc.task_handlers.base_task_handler.BaseTaskHandler(task_ctx=None, logger=None, **kwargs)[source]¶

Bases: object

An abstract base class to help with common task_handler logic.

Provides utility methods:
  • tick_task class method: instantiates class, sets self.task_ctx
  • validate_task_ctx
  • validate_task_params
Provides shortcuts:
  • self.task => task_ctx[‘task’]

Subclasses should at least implement initial_tick method.

class mc.task_handlers.switch_task_handler.SwitchTaskHandler(task_ctx=None, logger=None, **kwargs)[source]¶

Bases: mc.task_handlers.base_proxying_task_handler.BaseProxyingTaskHandler

Task handler that performs a conditional switch.

task_params:

control_value: value to use when checking cases cases: list of case dicts, that look like:

{'condition': condition_dict,
 'task': task to run if condition matches}

where a condition_dict looks like:

{'op': <operator from the python 'op' library  e.g 'eq'>,
 'arg': 2nd arg to operator}

default_case: the case to use if no case matches.
